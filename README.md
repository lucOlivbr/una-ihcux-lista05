No código foram aplicadas três heurísticas principais: status, prevenção de erros e estética.

A de status aparece quando o sistema mostra mensagens como “conectando” e “calculando”, assim o usuário sabe o que o programa está fazendo.

A prevenção de erros foi feita usando try-catch, que impede o programa de quebrar caso o usuário digite algo errado, mostrando uma mensagem de erro.

A estética foi aplicada deixando o sistema simples e organizado, com uso de cores e separadores, sem muita informação desnecessária.
